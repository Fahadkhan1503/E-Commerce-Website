# 🛍️ E‑Commerce Product Catalog

A simple product listing website that fetches data from **Contentful** CMS. No user login, no cart, no payment – just a catalog.

**Live:** [fahadroyalstore.netlify.app](https://fahadroyalstore.netlify.app)

## Features

- Product grid with image, title, price, description
- Product detail view
- Fully responsive (Tailwind CSS)
- Content managed via Contentful

## Tech Stack

- React + Vite
- Contentful (headless CMS)
- Tailwind CSS
- Axios
- Netlify
